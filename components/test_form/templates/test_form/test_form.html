Как сделано:

1) Предварительное заполнение поля ввода (placeholder) и первичная установка автофокуса заданы в классе формы в словаре
render_kw={'autofocus': True, 'placeholder': "+7(___)___-__-__" }

В этом словаре можно задавать различные атрибуты, которые обычно задаются в теге ввода данных input. Подробнее см.
http://programmerbook.ru/html/input/type/tel/
http://htmlbook.ru/html/input/type
https://xakep.ru/2018/09/24/wtforms/
https://atuin.ru/blog/maska-dlya-vvoda-nomera-telefona/

Placeholder и автофокус можно также задать в HTML внутри
{{ form1.user_phone(class="form-control", placeholder="+7(___)___-__-__", autofocus="True") }}
Результат будет один и тот же!

Если вместе с использованием плагина использовать атрибут value="..." - значение по умолчанию (либо в классе формы либо в html с использованием wtforms (в скобках)) то этот аргумент игнорируется (как я поняла). Если плагин не используется, то при задании аргумента value="..." его значение будет в поле ввода и при отправке пустой формы будет передано на сервер(в отличии от placeholder)).

2) Далее, для верного заполнения номера, используется плагин maskedinput jquery.
С его помощью создается маска заполнения номера телефона (например) и его валидация (например заполнение только цифрами).

В классе формы использую валидаторы RegExp и дополнительный валидатор в функции def validate_user_phone.
При использовании плагина maskedinput jquery это не нужно, в нем валидация есть!
Но я оставлю. Их можно использовать, если плагин не задействован.

3) Кроме того используется скрипт, который при клике на поле ввода устанавливает курсор на определенную позицию. У нас на 3(то есть после первой скобки)

Для использования maskedinput jquery необходимо:

1. Подключить jquery
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>

2. Подключить плагин maskedinput jquery
<script src="https://cdnjs.cloudflare.com/ajax/libs/
jquery.maskedinput/1.4.1/jquery.maskedinput.js" type="text/javascript"></script>

3. Подключить скрипт создания функции setCursorPosition установки курсора при клике

<script type="text/javascript">
$.fn.setCursorPosition = function(pos) {

if ($(this).get(0).setSelectionRange) {
$(this).get(0).setSelectionRange(pos, pos);
} else if ($(this).get(0).createTextRange) {
var range = $(this).get(0).createTextRange();
range.collapse(true);
range.moveEnd('character', pos);
range.moveStart('character', pos);
range.select();
}
};
</script>

4. Подключить скрипт maskedinput jquery с маской и дополнительной функцией setCursorPosition, заданной ранее
Обратите внимание что id объекта в скрипте ниже - это имя поля(?) в форме!!

<script type="text/javascript">
$("#user_phone").click(function(){
$(this).setCursorPosition(3);
}).mask("+7(999) 999-99-99");
</script>

Bootstrap Form Helpers - это набор плагинов jQuery, предназначенных для внешней среды Bootstrap. О помощниках формы начальной загрузки помощники по формам начальной загрузки - это коллекция плагинов JQuery, чтобы помочь вам создавать лучшие формы. Плагины могут использоваться по отдельности, но некоторые из них работают вместе, например, Страны и Штаты.

Коллекция включает в себя:

Пользовательский элемент выбора, Слайдер, Палитра цветов, Выбор страны, Государственный сборщик, Выбор валюты, Телефонный вход, Ввод числа, Средство выбора языка, Выбор часового пояса, Выбор шрифтов Google, Выбор даты, Сборщик времени, Выбор шрифта, Выбор размера шрифта

Плагин поддерживают все основные браузеры, но официально не поддерживают браузеры на платформах Linux. Требуется jQuery 1.10.2 или выше и Bootstrap 3.0.0 или лучше.

3) Условия, кот. накладываем на данные внутри функции, задающей дополнительную валидацию в классе формы

Имя функции при этом должно быть таким:
validate_имя валидируемого поля.
Это валидатор, который будет добавляться к встроенным валидаторам, заданным в форме. Пример:

class ApplicationForm(FlaskForm):

# В словаре render_kw={....} можно задать атрибуты поля,
# кот.(без Flask-wtform) можно задать в теге input
user_phone = TelField("Введите телефон",
[InputRequired(),
Regexp("^(...)*$",
message='Неверный формат телефона...')],
render_kw={'autofocus': True, 'placeholder': "+7(___)___-__-__" })

def validate_user_phone(form, field):
if field.data:
num = [int(i) for i in field.data if i.isdigit()]
if len(num)>11:
raise ValidationError('Телефон должен быть не более 11 цифр')
if len(num)<10:
raise ValidationError('Телефон должен быть не менее 10 цифр')
