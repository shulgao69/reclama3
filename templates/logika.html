Корзина

Корзина работает на основе сессии. В рамках одной сессии может быть несколько клиентов. Например на одном ПК может зайти несколько пользователей с разными пародями. Их корзины будут различны.

Сессии

Просмотреть все сессии можно print(session)
Получим словарь
Пример: {'_fresh': True, '_id': '47b3eda5752ce2eafef....', '_user_id': '1'}
Создать запись в сессии: например session['card_usluga_id']=35
Получить значение из сессии: например session.get('_fresh', '')
Очистить всю сессию: session.clear()
Удалить запись в сессии: session.pop('card_usluga_id')

Сессии, используемые на сайте:
Сессии, которые создаются автоматически:
- '_fresh': True, - показывает новая сессия или нет
- '_id': '47b3eda5752ce2eafef....' - это id сессии
- '_user_id': '1' - (создаются при авторизации пользователя (с помощью Flask-login?). Если пользователь не авторизован '_user_id' в словаре сессии нет
- 'csrf_token': '90e9a795eaa69cf1a1a8836936538863f99fa73e' - используется при создании форм? Когда создается?
Сессии, которые создаются НЕ автоматически:
- 'card_usluga_id': 39 -
- 'cart': [{'card_usluga_id': 38, 'count': 1, 'i': 1, 'j': 1, 'order_request_sum': '2000.45', 'price_id': 82}] -
- 'carts_users': [{'cart': [{'card_usluga_id': 38, 'count': 1, 'i': 1, 'j': 1, 'order_request_sum': '2000.45', 'price_id': 82}], 'user_id': 1}, {'cart': [], 'user_id': 'anonymous'}] - список корзин всех пользователей в рамках данной сессии
